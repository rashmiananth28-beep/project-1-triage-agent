\# Intelligent IT Triage \& Auto-Resolution Agent



Automatically classifies, resolves, or escalates IT support tickets using Claude AI.



\## Tech Stack

Python · FastAPI · Claude API · Freshdesk API · SQLite



\## Features

\- AI-powered ticket severity classification

\- Scored keyword search across knowledge base

\- Auto-resolution or escalation with findings

\- Token usage and cost tracking per API call

\- API key authentication on all endpoints



\## Setup

1\. Clone the repo

2\. Copy `.env.example` to `.env` and add your API keys

3\. Install dependencies: `pip install -r requirements.txt`

4\. Run: `python main.py`

5\. Open: `http://localhost:8000/docs`
